8 Puzzle Game
An interactive implementation of the classic 8 Puzzle Game, where the goal is to arrange the numbered tiles in order by sliding them into the empty space.
This project demonstrates concepts of problem solving, algorithms, and game logic.

Problem Description
The 8 Puzzle consists of a 3×3 grid with 8 numbered tiles and one empty space.
The player can slide tiles up, down, left, or right into the empty space.

Goal
Arrange the tiles in the following order:
1 2 3
4 5 6
7 8 _

Where _ represents the empty space.

Features
Shuffle the puzzle board
Move tiles using valid moves
Check for puzzle completion
Interactive gameplay
Displays number of moves taken
